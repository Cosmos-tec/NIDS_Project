Network Intrusion Detection System

Project Description

My Project is a Networking based project, the project goal is to identify possible networking attacks such as

  • ping of Death
  • TCP Syn flood attack
  • Fragmentation attacks

and to be able to notify system admins about possible breach in their network.

My NIDS functions mainly on the Transport Layer on the OSI model.

The system will scan all incoming traffic to it's connected server interface and parse packets that are being

sent to the server via tcp/ip, udp/ip & icmp while filtering for corrupted packets.
